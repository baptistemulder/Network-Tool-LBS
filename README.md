# Networking CRM v2

Personal networking CRM built for LBS MFA recruiting — Private Debt & LevFin London.

## Stack
- React 18 + Vite
- Anthropic API (message generation + profile analysis)
- Gmail MCP (draft creation)
- localStorage for persistence

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Hosted on Vercel. Push to main → auto-deploy.
